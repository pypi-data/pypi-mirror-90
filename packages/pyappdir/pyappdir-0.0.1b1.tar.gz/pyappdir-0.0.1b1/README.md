# python应用

## 安装
```shell
[编译版本]pip install .
[测试版本]pip install -i https://test.pypi.org/simple/ pyappdir
[发行版本]pip install pyappdir
```

## 运行
``` python
python -m app #默认运行
```

## 卸载
```
pip uninstall pyappdir
```
