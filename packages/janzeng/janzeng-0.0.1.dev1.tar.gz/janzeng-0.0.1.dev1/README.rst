Janzeng
=======

Janzeng

Documentation
-------------

Documentation is located at https://janzeng.readthedocs.io/

Features
--------

Minimal engine to dispatch events, entity system and sprites.

Installation
------------

Install janzeng by running:

    pip install janzeng
